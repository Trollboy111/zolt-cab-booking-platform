# Zolt Cab Booking Platform

Microservices-based cab booking platform built with Node.js, external APIs, and event-driven architecture.

## Live Demo
https://zolt-2hm4.onrender.com/

> Note: Backend services are hosted on Render free tier and may take 1–2 minutes to wake up on first request.

## Features

- Microservices architecture
- API Gateway integration
- Real-time booking workflow
- External API integration
- Event-driven communication
- Scalable backend services

## Tech Stack

- Node.js
- Express.js
- REST APIs
- Microservices
- JavaScript

## Architecture

The platform is divided into independent services connected through an API gateway and event-driven communication patterns.
